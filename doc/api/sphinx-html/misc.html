scitools.easyviz.misc¶

scitools.easyviz.misc.arrayconverter(a)¶
Convert the numpy array a with type ‘numpy.ndarray’ into a Numeric.array object (type ‘array’). This is useful when a ‘numpy.ndarray’ object is not accepted, which is the case in the gnuplot module.
